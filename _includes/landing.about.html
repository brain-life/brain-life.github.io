An online platform for reproducible neuroscience.

Apps

Share your analyses on brainlife.io by registering a GitHub.com repository as an "App." Apps can privately or publicly shared and used for scientific projects on the cloud computing platform.

Data

Share your neuroimaging data either privately or with the wider neuroscience community. Data on brainlife.io is organized as datatypes to allow reuse across projects and interoperability between Apps.

Computing

Share your computing resources privately with collaborators or publicly with the brainlife.io community. On brainlife.io you can register a resource to manage data and analyses locally or share your computers' contribution to advancing science and society.




How does it work?

Projects

Projects allow manage your data and analyses. Projects can be shared either with a selected number of collaborators or the entire community.

Brainlife.io manages data by mapping multiple files into registered as Datatypes. Data can be uploaded either using a web browser or the Brainlife Command Line Interface (CLI) for a bulk upload. When possible, brainlife.io will automatically match and convert your data into Datatypes during upload.

Bulk Processing

Our unique analysis management mechanism (pipeline) allows easy submition and monitoring of thousands of analyses.

Brainlife.io Apps are modular and each subject is processed independently. This leads to a high scalability, and makes Brainlife.io Big Data ready.


Visualize

Each datatype provides a set of visualizers. They include web based visualizers as well as native visualizers such as fsleyes, freeview, mrview, streamed to your browser via a brainlife.io novel mechanism.

Share your visualization tool to allow brainlife.io users to use it for research. more


Data provenance

Brainlife.io keeps track of the history of each dataset. It preserves input and output data files, Apps versions and configuration parameters as well as the computer resource used to generate the dataset.

You can count on brainlife.io to make sure your scientific results are reproducible.


Publication

Publish your full workflow, data, derivaties along with all the Apps used in a project with brainlife.io innovative publication method. A single Digital Object Identifyier (DOI) can be assigned to a full-record in a brainlife.io Project. Brainlife.io publications are archived securely with double redundancy for long term preservation.
