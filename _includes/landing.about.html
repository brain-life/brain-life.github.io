An online platform for reproducible neuroscience.

Apps

Share your analyses on brainlife.io by simply publishing the code on GitHub.com and registering it as a brainlife.io "App." Apps can privately or publicly shared and used for scientific projects on the cloud computing platform.

Data

Share your Neuroimaging data either privately with your collaborators or publicaly with the wider community. Data on brainlife.io is organized into datatypes to allow reuse across projects and interoperability between Apps.

Computing

Privately use your computing resources to run trusted Apps on your private data or share your newest cluster of cloud architecture with the wide brainlife.io community to show its impact on scientific discovery. brainlife.io allows registering a resource privately within collaborators of a project or openly across projects to increase resources utilization and prominance.




How does it work?

Projects

Project is used to store all your data and analysis. Project can be shared with your collaborators or made public.

Datasets on Brainlife are organized and regsitered as Datatypes. Data can be uploaded either using a web browser or the Brainlife Command Line Interface (CLI) for a bulk upload. Brainlife.io will automatically match and convert your data into known Datatypes, when possible.


Run App

See Apps for currently registered (public) Apps. To run an App, first create a new process which is used to stage the input data and store generated data derivatives. You will need to archive generated output if you want to preserve your outputs.

You can also use the Execute button under an App detail page to quickly launch the App. more


Bulk Processing

Our unique workflow management mechiansm (pipeline) allows you to easily submit and monitor thousands of tasks.

Brainlife Apps are modular and each subject is processed independently. This leads to a higher scalability, and Brainlife is ready for Big Data.


Visualize

Each datatype provides a set of visualizers. They include web based visualizers as well as native UIs (fsleyes, freeview, mrview) that can be streamed to your web browser using our novel mechanism.

If you are a developer of a visualization tool, we can make it come alive by allowing users to launch your apps on any dataset hosted on Brainlife. more


Data Provenance

Brainlife keeps track of how each dataset is generated. It preserves information such as the version of the App, configuration parameters and the resource used to generate the dataset.

You can count on Brainlife to make sure your data derivatives will always be reproducible.


Publish

Publish datasets and data derivaties along with all the Apps used to generate them under a single DOI. The published data will be archived on our secure tape archive for the long term curation.

Our publication mechianism allows your collaborators to easily reuse your research products.
